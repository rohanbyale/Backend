# Contributing to Free APIs Collection

Thanks for your interest in contributing! This project thrives on community additions and corrections.

## How to add a new API

1. Fork this repository.
2. Open `README.md` and find (or create) the relevant category section.
3. Add a new row to the table in this format:

   ```
   | API Name | Short description (max ~80 chars) | Auth (No/Key/OAuth) | HTTPS (Yes/No) | Link |
   ```

4. Keep entries in alphabetical order within their category where possible.
5. Make sure the API:
   - Is publicly accessible (no invite-only/private beta).
   - Has a free tier (doesn't need to be entirely free, but must offer free usage).
   - Has working documentation linked.
6. Open a Pull Request with a clear title, e.g. `Add: OpenWeather Air Pollution API`.

## Reporting a broken link or outdated entry

Open an issue with:
- The API name and category.
- What's wrong (dead link, no longer free, requires auth now, etc.).
- A suggested fix if you have one.

## Style guidelines

- Descriptions should be concise and factual — no marketing language.
- Use sentence case for descriptions.
- Don't include affiliate links or referral codes.
- One API per row; don't bundle multiple services into a single entry.

## Code of Conduct

Be respectful and constructive. We're all here to build a useful, accurate resource together.
