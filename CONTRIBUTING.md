# Contributing

Want to add a server to this list? Pull 

## Guidelines

1. **One entry per line/row**, placed in the most relevant category table.
2. **Format:** `[Name](link) | One-sentence description of what it does`
3. **Keep descriptions factual and short** — what the server does, not marketing copy.
4. **Verify the server works** before submitting — link to the actual source repo, not a placeholder.
5. If a server fits multiple categories, pick the primary one rather than duplicating it everywhere.
6. New categories are fine if you have 3+ servers that don't fit existing ones — open an issue to discuss first.

## Adding a new category

Add a new `##` heading in `README.md`, update the Contents list at the top, and include at least a couple of servers to start the section.
